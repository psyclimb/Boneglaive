# Boneglaive Patch Notes

---

## v1.3

### New Unit: LANDSCAPER

A four-armed terrain manipulator who reshapes the battlefield through acoustic resonance.

### New Map: Verdant Terrace

A jungle patio with flagstone paving at its center and dense undergrowth at the edges.

### Balance

**FOWL CONTRIVANCE**
- Gaussian Dusk no longer destroys terrain/furniture at the beam's midpoint

### Bug Fixes

**FOWL CONTRIVANCE**
- Parabol Underground's imprint effect now correctly relocates dynamic terrain (slag walls, marrow dike walls, derelict buildings, topiaries) without breaking their durations

**DERELICTIONIST**
- Fixed Derelict skill's decrepit building spawning without the upgrade

**DELPHIC APPRAISER**
- Market Futures anchors are now properly removed when the imbued furniture is destroyed or overwritten by another skill
- Parallax status is now correctly cleared from adjacent units when the anchor they were next to is destroyed

**INTERFERER**
- Fixed AI not correctly evaluating Scalar Node trap placement, leading to suboptimal or broken trap positioning

### UI

- Overhauled menu screens with consistent styling, panels, and shared animated background
- Added custom terrain texture for Derelict building tiles
- Fixed version number showing "v0.9.0b BETA" on the text mode about screen and title
- Fixed window title showing "Boneglaive Beta" instead of "Boneglaive"
- Removed tile highlighting during unit placement and respawn selection

### Internal

- Removed deprecated LAN/network multiplayer code
- Removed deprecated SimpleAI system
- Removed duplicate skill definitions
- Removed stale development documentation
- Cleaned up dead code, unused imports, and debug artifacts throughout the codebase
- Renamed `network_mode` config key to `game_mode` for clarity
- Fixed version number mismatches across package metadata
- Adopted project-wide code standard
