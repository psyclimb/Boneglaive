# Boneglaive Patch Notes

---

## v1.1

### Balance

**DERELICTIONIST**
- Difficulty rating increased: ★★★ → ★★★★
- Base attack reduced: 6 → 0
- Basic attack damage now scales with chess distance to target
- Basic attacks now bypass defense
- Severance passive now activates on basic attack, not just skills
- Derelict healing increased by 70% — now heals 1–15 HP based on final distance
- Partition dissociation cooldown penalty reduced: 8 → 6
- Partition can now be self-cast without requiring the upgrade
- Partition (upgraded): upgrade now only removes the dissociation cooldown penalty (self-cast no longer exclusive to upgrade)

**DELPHIC APPRAISER**
- Attack reduced: 5 → 4
- Attack range increased: 1 → 2

**GRAYMAN**
- Basic attacks no longer bypass defense
- Estrange damage increased: 3 → 4
- Estrange no longer bypasses defense
- Estrangement status effect now expires after 3 turns (was indefinite)

**GRAYMAN DOPPELGANGER**
- Basic attacks no longer bypass defense
- Attack increased: 3 → 4
- Death explosion damage increased: 3 → 6

**MARROW CONDENSER**
- Difficulty rating reduced: ★★★★ → ★★★

### AI

- DERELICTIONIST AI uses distance-based Severance attacks with optimal post-attack repositioning
- DERELICTIONIST AI scores attack actions using actual expected damage rather than base attack stat

### Maps

**The Lime Foyer Arena**
- Replaced 8 limestone tiles with dust floor at the left and right mid-field flanking positions

### UI

- Unit info panel now displays position coordinates instead of unit ID
- Clicking floor and terrain tiles now shows an info panel
- Removed "Requires DELPHIC APPRAISER" hint text from furniture info panel
- Severance descriptions swapped between setup preview and advanced help page
- Issuing an attack now grays out the Move, Attack, and Skills buttons (consistent with skill behavior)
- Moving a unit no longer auto-enters attack mode — players must press Attack manually
- Execute Turn button is now always green
- Help and Execute Turn buttons swapped in the action menu
- HELP button renamed to UNIT HELP
- Unit help page now blocks clicks to the game board while open
- Unit help page can be closed with right-click

### Help Pages

- Added Aerosolize Arms entry to GAS MACHINIST help page and setup preview
- Added Parallax and Deft(?) Reroll entries to DELPHIC APPRAISER help page and setup preview
- Corrected Market Futures cooldown displayed on help page: 6 → 4 turns
- Clarified Market Futures and Parallax descriptions — teleport and Investment effect now correctly attributed to Parallax
- Renamed teleportation "anchor" to "locus" across all player-facing text

### Bug Fixes

- Fixed skill bar not displaying after issuing a move then pressing S in the graphical version
- Fixed HEINOUS VAPOR units displaying as "HEINOUS VAPOR" in the unit info panel — they now display their specific name (e.g. BROACHING GAS, CUTTING GAS, LIVING AEROSOL)
- Fixed GAS MACHINIST losing Aerosolize Arms skill on death and respawn

### Other

- Version number updated to v1.1 on main menu and about screen
