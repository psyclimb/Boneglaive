# Boneglaive Patch Notes

---

## v1.2

### Visuals

- Added tile graphic for the Derelict building

### Skill Changes

**DERELICTIONIST**
- Derelict building now places over existing terrain and furniture, restoring them when the building expires
- Derelict building now forms whenever Derelicted is applied, including from the base Derelict skill, Vagal Run at long range, and Partition dissociation

**POTPOURRIST**
- Selenic Backdraft zone now checks the position the attacker queued their attack from, preventing units from moving out of the zone then attacking

### Skill Changes

**FOWL CONTRIVANCE**
- Rail Genesis death explosion no longer damages friendly FOWL CONTRIVANCE units

### Balance

**DELPHIC APPRAISER**
- Base attack reduced: 4 → 3
- Market Futures cooldown increased: 4 → 5

### UI

- Fixed text and buttons overlapping in the minimized game over bar
- Fixed Estranged status effect not showing remaining turns in the unit info panel
- Fixed Deft(?) Reroll icon not appearing on the skill bar
- Removed internal respawn prompt messages from the combat log

### Bug Fixes

**MARROW CONDENSER**
- Fixed Bone Tithe only hitting one unit at range 2 — AoE now correctly covers the full 5x5 area instead of 8 directional spokes
- Fixed Bone Tithe not doubling HP gain per hit when upgraded by Dominion
- Fixed Partition's +1 PRT persisting indefinitely after the shield was removed by Vagal Run or caster abandonment
- Fixed Site Inspection stat bonuses not being removed when cleansed by Vagal Run
