# Numpad Control Scheme

An ergonomic alternative control scheme for one-handed gameplay using the numeric keypad.

## Action Menu Controls

```
┌─────────────────────────────────┐
│        NUMPAD LAYOUT            │
├─────────────────────────────────┤
│                                 │
│  [7] Help      [8] ↑       [9] Skills │
│                                 │
│  [4] Move      [5] -----   [6] Attack │
│                                 │
│  [1] Respawn   [2] ↓       [3] Upgrade│
│                                 │
│  [0] Execute Turn    [.] Concede│
│                                 │
└─────────────────────────────────┘
```

### Action Keybinds

| Numpad Key | Action | Description |
|------------|--------|-------------|
| `7` | Help | Open help screen |
| `9` | Skills | Toggle skills panel |
| `4` | Move | Enter movement mode |
| `6` | Attack | Enter attack mode |
| `1` | Respawn | Open respawn menu |
| `3` | Upgrade | Open upgrade menu |
| `0` | Execute | Execute queued actions |
| `.` | Concede | Concede the match |

## Skill Controls

```
┌─────────────────────────────────┐
│      SKILL SHORTCUTS            │
├─────────────────────────────────┤
│                                 │
│  [/] Skill 1   [*] Skill 2      │
│  [-] Skill 3   [+] Skills 4-6   │
│                                 │
└─────────────────────────────────┘
```

### Skill Keybinds

| Numpad Key | Skill | Description |
|------------|-------|-------------|
| `/` | Skill 1 | Activate first skill |
| `*` | Skill 2 | Activate second skill |
| `-` | Skill 3 | Activate third skill |
| `+` | Skills 4-6 | Cycle through skills 4, 5, and 6 |

**Note:** Press `+` multiple times to cycle through skills 4, 5, and 6.

## Design Philosophy

- **Most frequent actions** (Move, Attack, Execute) positioned for easy thumb/finger reach
- **Dangerous actions** (Concede) placed away from common keys to prevent accidents
- **Logical spatial grouping** mirrors the on-screen UI layout
- **One-handed operation** - all controls accessible with right hand on numpad
- **Original keybinds preserved** - both control schemes work simultaneously

## Traditional Controls

All original keyboard shortcuts (M, A, S, U, R, T, H, C for actions; 1-4, Q, W for skills) remain fully functional.
