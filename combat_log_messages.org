* Notes

** Message Formatting
- Messages prefixed with #HEAL_# indicate healing values shown in green
- Variables in curly braces are replaced at runtime with actual values
- Some messages appear incomplete in the source (marked with "incomplete in source")
- Messages may have variants for upgraded vs base skill versions

** Common Patterns
- Immunity messages reference "Stasiality" (GRAYMAN's passive)
- Position coordinates use (y, x) format
- Player colors and names are dynamically inserted
- Combat damage messages use add_combat_message() helper
- Ability activation messages use add_message() with MessageType.ABILITY

** Technical Details
- Messages extracted from: boneglaive/game/skills/*.py
- Total message count: ~213 distinct message_log calls
- Each unit has multiple skills with unique messages
- Messages support both ASCII and graphical game modes
